Co-Write（コライト）とは

**本プロジェクト（Co-Write：コライト）は、「リビングラボ」に関する書籍づくりのプロジェクトです。**日本の社会文化的な文脈におけるリビングラボ方法論の構築をめざす研究を進めている「[LLL（Living Lab Laboratory）](https://livinglablaboratory.com/aboutus/)」が主催しています。

Co-Writeは、単なる書籍づくりのプロジェクトではありません。「**リビングラボ」に関する書籍を「共創的に」制作していくプロジェクトです**。ここでのリビングラボは、ソーシャルイノベーションに向けた「共創」を行うための環境や方法論のことをさします。つまり、本プロジェクトは、**「共創のための書籍を共創的につくる」という実験的な取り組み**なのです！

「リビングラボ」という言葉は、近年、日本の中でも普及してきました。我々がリビングラボに関する研究を始めた当初（10年前くらい）は、リビングラボという言葉を知っている人は殆どいませんでした。ところが今では、リビングラボを知っている、関心がある人が、非常に多くなってきたように思います。一方、リビングラボは、非常に多様なアプローチを含む概念です。リビングラボと一言で言っても、新興テクノロジーの開発を重視した取り組みもあれば、地域のコミュニティ開発を重視した取り組みもあります。企業が牽引する取り組みもあれば、市民が主導する取り組みもあります。そのため、リビングラボは、捉えどころのない曖昧な概念だと言えます。加えて、日本には、リビングラボの概念や進め方について詳細に解説する資料や情報が、これまで殆どありませんでした。そのため、**「リビングラボとは何か？」ということの本質的な部分について、曖昧な理解や偏った理解がされてしまっていることも多い**ように見受けられます。

そこで我々LLLでは、特に、**ソーシャルイノベーションという文脈から、「リビングラボ」の考え方や運営方法の本質、国内外のよい事例を紹介するための書籍やガイドブックを作成するプロジェクト**を開始しました。それが、本プロジェクトの始まりです。

一方、リビングラボの根幹は「共創」です。そのため、書籍をつくるプロジェクトの中で、「共創のための書籍を従来的な（著者と編集者に閉じた）クローズドなアプローチで作成していいのだろうか……？」という疑問が我々の中に生まれました。我々が伝えたいことの「コア」は表現しつつも、様々な立場の人や専門性を持つ人の視点や知見、考えを受け取り、それを反映させていくことで、我々だけではつくれないような書籍ができるかもしれません。また、書籍を共創的につくる過程で、様々な議論が生まれたり、同じ志を持った仲間がさらに増えたりすれば、その過程（プロセス）自体に、（書籍という単純なアウトプットを超えた）価値が生まれます。そして何よりも、共創のための書籍は、共創的につくるべきです。このような考えのもと、我々は、**リビングラボの書籍を共創的につくるためのプロジェクト＝Co-Write**を開始しました。

# 書籍について

本プロジェクトで制作しようとしている書籍**『リビングラボ：ソーシャルイノベーションのための共創のプラットフォーム（仮）』**は、**リビングラボを、「ソーシャルイノベーションのための共創を支援・加速するための仕組み（場・環境）」と位置づけ、その特徴や価値、メカニズムをわかりやすく紐解いていく書籍**です。特に以下の点について、LLLメンバの研究や実践経験から得られた知見を解説しています。

<aside>

- リビングラボとは何か？　なぜ、ソーシャルイノベーションのためのアプローチとして有効なのか？
- ソーシャルイノベーションのためのリビングラボは、どのような構造を持った仕組みなのか？　いかにして、ソーシャルイノベーションのための共創を支援・加速するのか？
- リビングラボは、他の類似したアプローチと比較して、どのような価値（メリット）や難しさがあるのか？　また、リビングラボというアプローチ固有の、本質的な価値とは何なのか？
- リビングラボを持続的かつ効果的に運営するためには、どのような実践・マネジメントのアプローチがあるのか？
- 日本の社会文化的文脈の中で、リビングラボを立ち上げ、運営するためには、どうしたらいいのか？

# あなたができること

皆さんは、**LLLメンバで作成している書籍『リビングラボ：ソーシャルイノベーションのための共創のプラットフォーム（仮）』の内容に対して、ご自身の意見や考えを、コメントの形式で提案することができます。**単純な文章の修正やブラッシュアップに関するコメントはもちろんのこと、「ここは違うんじゃない？」という指摘や、「もっとこういうことも書いてほしい！」という要望をコメントすることもできます。そして、いただいたコメントを参考に、LLLのメンバーが、書籍の内容（本文）を変更・更新していきます。このように、皆さんは、**本書籍の共同編集者（Co-Writer：コライター）となり、日本の様々な実践家や研究者にとって役立つ書籍を、一緒につくりだしていく一員となります。**

書籍へのコメントは、このNotion上で行います。詳細な利用方法は、「[共同執筆の仕方](https://www.notion.so/25bdba2676028061b314c127ffe14b0c?pvs=21) 」をご覧ください。なお、いただいたコメントにもとづいて、どのように本文の内容を修正・更新するかは、LLLのメンバーで議論して、決めさせていただきます。なぜならば、修正・更新の方針は、書籍全体の一貫性やバランス、LLLとして伝えたいことをべースに検討しなければならないためです。そのため、必ずしもすべてのコメントが反映されるわけではない、かもしれません。ただし、本プロジェクトの一番の目的は、みんなでリビングラボの書籍を共創することです。そのため、**できるだけ多くのコメントを反映させていきたい**と考えています。

**コメントをくださった方のお名前（ニックネーム）は、書籍が発行される際に、「Co-Writer」として記載させていただく予定です**。ただし、書籍の発行については、いくつかの出版社の方々と既に相談はしていますが、まだ決定している訳ではありません。その意味では、本プロジェクトで共同作成した成果物を、そのまま出版社から発行する、自費出版する、オープンアクセスなデジタル書として発行する、プロジェクト終了後に改めて加筆修正したものを出版する、などなど、様々な可能性が残されています。我々も、どのような展開の仕方がふさわしいのか、まだ決めかねている段階です。いずれの場合にせよ、どのような方々に貢献していただいたかは、（ご本人の同意のもとで）明確に示せるようにしたいと思います。

本プロジェクトは、**多様な人に参加していただきたい共創型のプロジェクト**です。必ずしも、リビングラボや共創の実践が豊富な方の専門的なコメントばかりが欲しいわけではありません。リビングラボの実践経験がない方、学生さん、ちょっと興味がある方、などの参加も歓迎します。コメントの内容についても、「ちょっと気になったことを書いてみた」というコメントも大歓迎です。どんな書籍をつくっているんだろう？という興味からまずは参加していただくのも、もちろん問題ありません！　**ぜひ、お気軽な気持ちでご参加ください！　幅広い多様な方々の参加をお待ちしております！**
