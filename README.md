**本プロジェクト（Co-Write：コライト）は、「リビングラボ」に関する書籍づくりのプロジェクトです。**日本の社会文化的な文脈におけるリビングラボ方法論の構築をめざす研究を進めている「[LLL（Living Lab Laboratory）](https://livinglablaboratory.com/aboutus/)」が主催しています。
Co-Writeは、単なる書籍づくりのプロジェクトではありません。「**リビングラボ」に関する書籍を「共創的に」制作していくプロジェクトです**。ここでのリビングラボは、ソーシャルイノベーションに向けた「共創」を行うための環境や方法論のことをさします。つまり、本プロジェクトは、**「共創のための書籍を共創的につくる」という実験的な取り組み**なのです！
